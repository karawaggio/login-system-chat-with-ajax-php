## login-system-chat-with-ajax-php

A small chat project. Stack includes:

* MySQL for CRUD operations
* PHP to establish the connection with database, process and check data
* AJAX, jQuery to dynamically display content of the chat

In order to run the project go to functions.php and change server name, user name, password and database name related to your setup. Next change database name and sql queries to match your database information.
